# Softfile Diagram Editable SIMASET

Folder ini berisi file sumber diagram yang dapat diedit ulang.

| File | Keterangan | Aplikasi Edit |
| ---- | ---------- | ------------- |
| `ERD-SIMASET.drawio` | Softfile ERD dalam format diagrams.net/draw.io | diagrams.net / draw.io |
| `ERD-SIMASET.puml` | Softfile ERD dalam format PlantUML | PlantUML / VS Code PlantUML |
| `DFD-SIMASET.puml` | Softfile DFD dalam format PlantUML | PlantUML / VS Code PlantUML |

Catatan:
- File `.drawio` paling mudah diedit secara visual melalui diagrams.net.
- File `.puml` paling mudah diedit sebagai teks lalu diekspor ulang menjadi PNG.
