# Wireframe BAB V SIMASET

Folder ini berisi wireframe antarmuka SIMASET untuk bagian perancangan interface pada `BAB-V-SIMASET-DRAFT.md`.

Versi terbaru dibuat menggunakan HTML dan CSS saja agar struktur UI dapat disamakan dengan aplikasi asli.

## File Utama

| File | Keterangan |
| ---- | ---------- |
| `index.html` | Katalog seluruh wireframe halaman SIMASET |
| `styles.css` | Design system dan layout statis |
| `ANALISIS-UI.md` | Analisis UI aplikasi sebagai dasar wireframe |
